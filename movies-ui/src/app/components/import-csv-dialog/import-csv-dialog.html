Import comma-separated movies

Each line is one movie. Two formats are accepted, and you can mix both in the same file:

  • By IMDb id — just the id: tt0075314
  • By title and year — a blank id, then title, then a 4-digit year: ,Taxi Driver,1976

Extra trailing columns (e.g. a director) are read but ignored. Quote a title with a comma in it, e.g. ,"Ocean's Eleven",2001. Lines that don't have a usable id or a title with a valid year are skipped entirely — they never show up as failed.

Example file:
tt0075314,,
,Taxi Driver,1976
,"Ocean's Eleven",2001,Steven Soderbergh
