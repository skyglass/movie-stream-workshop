Import comma-separated movies

Each line is one movie. The first column is the IMDb id. Every column after that is optional — one quoted, dot-separated category path to file the movie under, e.g. "Genres.Drama". A movie with no category paths is assigned directly to this guide (or its selected sub-category).

Quote each category path so a comma inside it (part of a category name) doesn't get mistaken for a column separator, e.g. "Narrative, Art & Characters.German Expressionism". Lines whose first column isn't a non-blank imdb_id are skipped entirely — they never show up as failed.

Example file:
tt0075314
tt0468569,"Genres.Drama"
tt0246434,"Genres.Drama","Narrative, Art & Characters.German Expressionism"
